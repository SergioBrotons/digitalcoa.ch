Enforcement has matured past the principle-establishment phase into case-building methodology.

Carrier attestation format evolution has revealed documentation gaps that firms did not know they had. The implementation date creates an acute forcing function for the healthcare vertical and related professional services.

**UNAIWG December 2026:** Geneva international organizations (UNOG, WHO, WTO, ITU, UNHCR, ICRC) face UNAIWG AI accountability framework compliance deadline. **Federal Council consultation draft:** Expected in 2026. The consultation response architecture requires anticipatory positioning — firms that wait for publication before building documentation will face the same retroactive documentation problem that characterizes current nFADP exposure. The pre-publication window is the operative compliance window.
