1. **73% failure rate** — governance as failure-prevention
2. **GPAI ~8mo already operative** — EU-nexus firms already obligated
3. **40% skills gap** — advisory opportunity
4. **31-point belief-action gap** — 71% using AI without formal framework
5. **GPAI transparency obligations** — client disclosure requirements

## Objection Handling

### "We're not in the EU"
→ GPAI obligations apply to any consulting firm deploying EU-origin GPAI systems to EU clients or work product used in EU.
→ If any EU nexus exists, GPAI applies.

### "Our projects are small"
→ FADP applies regardless of project size.
→ GPAI obligations apply based on system deployment, not client size.

### "We failed because of technology"
→ 73% of failures are governance/skills, not technology.
→ This is a solvable problem.
