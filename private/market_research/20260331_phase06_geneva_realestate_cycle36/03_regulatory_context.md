## Regulatory Framework (Geneva Real Estate)

### Primary: FADP/nFADP — Already Operative
- **FADP Art. 5(c)** active since **Sep 1, 2023** — NO GRACE PERIOD
- FADP applies to AI-supported data processing without new legislation
- FDPIC expects **purpose + functionality + data sources** disclosure
- Criminal liability: **CHF 250,000** for intentional FADP failures
- **DPIA required** for high-risk AI processing (property valuation AI, tenant screening AI)

### AI Applications in Real Estate (FADP Exposure Points)
1. **Property valuation AI** — processing property data = likely high-risk
2. **Listing description generation AI** — transparency disclosure required
3. **Tenant screening AI** — creditworthiness assessment = high-risk (EU AI Act Annex III)
4. **Market analysis AI** — data processing + disclosure obligations
5. **Customer communication AI** — chatbot/AI disclosure required
